# Acceptance Criteria

## Phase 1 - Bronze Layer

### Complete When

- [ ] All source CSVs loaded into PostgreSQL
- [ ] Row counts validated
- [ ] No transformations applied
- [ ] ETL script automated
- [ ] Documentation updated

---

## Phase 1 - Silver Layer

### Complete When

- [ ] Missing values handled
- [ ] Duplicates checked
- [ ] Data types standardized
- [ ] Validation checks created
- [ ] Derived columns created

---

## Phase 1 - Gold Layer

### Complete When

- [ ] Fact tables created
- [ ] Dimension tables created
- [ ] Metrics tables created
- [ ] Power BI connects only to Gold layer
